# Config Scout Log

Comparison reports generated by `/config-scout` live here.

Each file is a timestamped comparison of an external Claude/agent configuration
against the local AlpheusCEF setup. Files follow the naming convention:

```
YYYY-MM-DD_<source-slug>.md
```

Re-running `/config-scout` against a previously analyzed source generates a new
dated report and includes a delta section showing what changed since the last
comparison of that same source.
